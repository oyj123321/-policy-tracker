# Policy Tracker 使用说明

此页面供 Claude Code 自动加载。当你在此目录下工作时，以下 prompt 模板可用。

## 生成周报（每周一）

> 搜索上周全球产业政策，按行业分中+国际汇总，保存到 `D:\news\周报-政策追踪-YYYY-MM-DD.html`（用本周一的日期）。报告结构：封面（政策周报+日期+覆盖周期）、中国产业政策（新能源/电动车、半导体/科技、房地产/基建/城市更新、消费/以旧换新、农业，每个子项用表格，列政策名+类型+金额/规模+有效期+要点+市场映射）、国际产业政策（美国CHIPS/IRA/半导体、欧盟CBAM/绿色工业/关键原材料、芯片出口管制/实体清单、全球电动车关税、日韩/东南亚科技投资）、专家拆解（挑2-4条本周最重要的政策，每条包含：为什么现在出、与中国/国际政策联动、历史参照、反对观点、后续演化）、下周关注（未来一周关键数据发布+央行会议+财报+论坛，用卡片式布局，红色标记最热事件）、本周值得关注的产业方向（2-3条，每条说明机会或风险）、来源分级（🟢官方一手/🔵权威媒体/⚪券商智库）。每个表格后加黄底斜体市场映射行。金额列不能写"—"，填具体数字。报告输出为纯HTML，样式参考 `gen_report_v2.py`。

## 生成日报

> 搜索过去24小时重大宏观事件，分中国线、国际线、关键资产价格、科技突破扫描四块，末尾附一句话总结。保存到 `D:\news\日报-YYYY-MM-DD.html`。HTML 样式：蓝底封面+表格斑马纹+彩色标签+专家拆解卡片。

## 生成下周前瞻（每周日）

> 搜索下周全球经济数据发布日历，列出下周一到周六的重要事件（日期+时间+事件+前值+预期+为什么重要+超预期意味+不及预期意味），挑1-2个焦点做深度拆解（含市场推演表格和情景分析），末尾附简单决策矩阵（情景→风险资产→避险资产→美元方向）。保存到 `D:\news\周报-前瞻-YYYY-MM-DD.html`。

## 搜索流程

每次生成须完成以下搜索（同组可并行）：

**第一轮（并行4路）**：
- 中国产业政策 最新（新能源/电动车/半导体/房地产/消费/农业，用当周日期）
- US CHIPS Act / IRA / semiconductor export controls latest
- EU CBAM / green industrial policy / critical raw materials latest
- 全球电动车关税 芯片管制 日韩半导体投资 最新

**第二轮（按需补充2-3路）**：
- 中国专项债发行进度 + 以旧换新补贴具体数据
- 下周全球经济数据发布日历（仅前瞻周报需要）
- 深度拆解所需的历史案例对比

**写作原则**：
1. 政策名+类型（新政/续期/加码/收紧）+ 金额 + 有效期 + 要点 + 市场映射
2. 金额列不能写"—"，必须填具体数字或解释为什么无法获取
3. 每条深度拆解含：为什么现在出 + 历史参照 + 反对观点 + 后续演化
4. 来源分级：官方一手公告 > 权威媒体报道 > 券商/智库分析
5. HTML 输出，纯手写，不通过 markdown 渲染
